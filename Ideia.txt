- Fabricante

   NAO DEVE EXISTIR FABRICANTES COM MESMO NOME NO BANCO (nao pode ser case sensitive)

   -> Cadastro ok
   -> Atualização ok
   -> Detalha (pega todas as infos) ok

   * Nome fabricante ok
   * Pais de origem (Brasil, china, etc) ok
   * Data Cadastro ok
   * Data Atualicao ok
